Prints **two 38×25 mm stickers per item** in a single print job:

1. **Spine label** — Library name header + call number (each segment on its own line)
2. **Barcode label** — Library name header + truncated title + barcode image

---

## Installation

1. Copy the `dual_label_print/` folder into your SLiMS `plugins/` directory:

```
slims9_bulian/
└── plugins/
    └── dual_label_print/
        ├── dual_label_print.plugin.php   ← plugin registration file
        ├── index.php                      ← main plugin page
        └── README.md
```

2. Log in to the SLiMS admin panel.
3. Go to **System → Plugin Manager** and activate **Dual Label Print**.
4. The menu item **"Dual Label Print"** will appear under the **Bibliography** module.

---

## Usage

1. Open **Bibliography → Dual Label Print**.
2. Search by accession number, title, or author in the search box.
3. Tick the checkboxes next to the items you want and click **"Add To Print Queue"**.
4. Repeat for any additional items (up to 50 at once).
5. Click **"Print Dual Labels"**.
   - A print preview window opens.
   - The browser's print dialog fires automatically after ~1.2 s (time for barcode PNGs to be written to disk).
6. Select your sticker printer (38×25 mm roll or sheet) and print.

## Label layout (per item)

```
┌────────────────────┐  ← page break
│   LIBRARY NAME     │  header (grey bar)
│  ────────────────  │
│  005               │  call number
│  SUP               │  (each space-delimited
│  2025              │   token on its own line)
└────────────────────┘  ← page break  (sticker 1 end)

┌────────────────────┐  ← page break
│   LIBRARY NAME     │  header (grey bar)
│  Title of Book...  │  truncated title (6pt)
│  ┌──────────────┐  │
│  │ |||||||||||  │  │  barcode image
│  │  0001234567  │  │
│  └──────────────┘  │
└────────────────────┘  ← page break  (sticker 2 end)
```

---

## Notes

- Barcode images are generated by SLiMS's built-in `lib/phpbarcode/barcode.php` and cached in `files/images/barcodes/`. Make sure that directory is writable.
- Print settings (fonts, border size, header text, barcode scale, title cut-off) are shared with the core **Labels Printing** and **Item Barcodes** modules. Change them via **Bibliography → Change print label settings** or **Change print barcode settings**.
- The `files/` (UPLOAD) directory must be writable so the HTML print file can be saved.
- Maximum 50 items per print job (same as core modules).
